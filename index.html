Data Satwa

Halaman untuk menampilkan data satwa yang ditangkar.

Data Satwa Di sini nanti bisa diisi tabel penangkaran per jenis satwa, jumlah individu, status perlindungan, dan informasi lainnya.
Data Mortalitas Di sini nanti berisi ringkasan dan tabel mortalitas satwa per periode, penyebab kematian, dan catatan medis.
Data Natalitas Di sini nanti berisi data kelahiran per jenis, periode, dan informasi induk/keturunan.
QnA Di sini dapat diisi daftar pertanyaan umum seputar pengisian data penangkaran dan panduan teknis.
Export Data Di sini nantinya berisi tombol untuk mengunduh data dalam format CSV/Excel atau terhubung ke Google Sheet.
Data Perusahaan Di sini nanti menampilkan daftar pemegang izin / perusahaan penangkar, kontak, dan informasi legalitas.
